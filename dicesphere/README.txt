# Summary

D6 (six-sided) die with the pips on a sphere. Overall dimensions: 16 mm x 16 mm x 16 mm.

The first file is for printing one die alone, and the second file is for printing four dice at once.

This is my first design, so please provide feedback about improvements to the design or the instructions!
